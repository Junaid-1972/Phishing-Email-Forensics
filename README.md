# Phishing-Email-Forensics
This analysis breaks down the email’s technical headers and content, highlighting indicators of compromise such as spoofed domains, suspicious URLs, and manipulative language. The outcome provides actionable insights into detection methods, user awareness, and defensive measures to strengthen organizational resilience against phishing attacks.

This project is inline with NIST Cybersecurity Framework (CSF) and also ISO 27001 Annex A Control(s):
- Detect (DE.AE – Anomalies and Events): Identifying suspicious email activity and anomalies.
- Respond (RS.AN – Analysis): Analyzing phishing incidents to support incident response.

- Annex A.8.17 Security Event Analysis
- Annex A.5.29 Incident Handling
